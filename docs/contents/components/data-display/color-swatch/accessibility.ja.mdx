`aria-label`を設定すると、スクリーンリーダーによって読み上げられます。

```tsx live=false
<ColorSwatch color="#4387f4" aria-label="Blue" />
```

## ARIAロールと属性

| コンポーネント | ロールと属性                          | 使い方                       |
| -------------- | ------------------------------------- | ---------------------------- |
| `ColorSwatch`  | `role="img"`                          | 画像であることを示します。   |
|                | `aria-roledescription="color swatch"` | `color swatch`を設定します。 |
